honorCode Teacher Academy Hackathon

This fun challenge will expose you to several extremely important concepts in software development.

  1. Importing external libraries
    • Bootstrap: A great CSS/JS library used to structuring and styling webpages, with many helpful components included.
    • jQuery: A lightweight JavaScript library with powerful function for HTML manipulation and Ajax handling.
  2. Calling APIs (Application Programming Interfaces)
    • This is how programs talk to each other!
    • The open web is filled with thousands!
  3. Where to look for help
    • This is the most important skill as a programmer: Learning how to search Google efficiently, read new language/library documentation, search StackOverflow, etc.

Over the course of the next several hours, you'll work in small teams to implement your own webpage that:

  1. Uses Bootstrap for page styling
  2. Uses jQuery for making HTTP calls to an external API
  3. Given a list of terms, displays a glossary with definitions fetched from an API
  4. Has the ability for a user to enter more words into the glossary, automatically fetch the definition, and add an entry to the table
